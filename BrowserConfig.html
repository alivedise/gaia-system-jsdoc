Class: BrowserConfig

BrowserConfig

new BrowserConfig()

This class generates browser config from manifestURL or src provided. If manifestURL is provided, we would treat it as a web app, and then use Applications to fetch the remaining info we need to construct the options for mozbrowser iframe.
